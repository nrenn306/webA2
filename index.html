Assignment 2: Web2

group members: Seila Kennedy & Nicole Rennie

This website is a simple online clothing store built as a Single-Page Application. Its goal is to let users browse products, view detailed information, and manage a small shopping cart, all without leaving the page. The site loads product information from a JSON data source and dynamically updates the displayed view using JavaScript.

    Technologies used:

  • HTML5
  • CSS
  • Vanilla JavaScript
  • JSON
